- **Decisão:** Utilizar o GitHub Pages como Dashboard principal em vez do Canvas nativo ou Google Drive.
- **Racional:** Maior estabilidade em servidores VPS, acesso universal via URL e facilidade de atualização autónoma via código.

- **Racional:** Otimização de recursos e organização lógica por tipo de media.
- **Status:** Ativo.
